My Tech Journey

From curiosity to professional development

The Beginning: SS Browser

My tech journey began in Class 7 when I created my first web browser called "SS Browser" using a third-party no-coding web application. This achievement, though built with simple tools, ignited a fire within me.

The excitement of seeing my own browser work was unparalleled. It taught me that with determination, even complex technological creations are possible.

The Evolution: Personal Website

Gradually, I moved from no-code tools to actual programming. I built my personal website shamsadshanin.my.id from scratch. This project taught me the fundamentals of web development.

Since then, I haven't looked back. I immersed myself in learning full-stack development, database management, and software architecture.

Where I Am Today

Today, I'm a confident full-stack developer with expertise in PHP, JavaScript, MySQL, Laravel, and React. I've built multiple production-ready ERP systems and management software solutions. Alhamdulillah, every step of this journey has been a blessing and a learning experience.
